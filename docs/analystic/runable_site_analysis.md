## 1. 사이트 개요 / 목적

러너블(runable.me)은 러너(달리기 동호인) 종합 플랫폼입니다. 운영 주체는 러너블(주)이며 중앙일보(joins) 계열로 보입니다(대표메일 runable@joins.com, 주소: 서울 마포구 상암산로 48-6 중앙일보빌딩, 사업자등록번호 360-81-02326).

핵심 목적 4가지:
1. **대회 플랫폼** — 국내·해외 마라톤/러닝 대회 검색 및 접수·결제
2. **기록 관리** — 개인 러닝 기록 수집·분석·시각화
3. **커뮤니티 & 콘텐츠** — 러너 커뮤니티와 매거진
4. **커머스** — 러닝 용품 스토어 및 해외 런트립(여행) 상품 판매

즉 "대회 접수 + 기록 관리 + 커뮤니티 + 커머스"를 묶은 올인원 러너 서비스입니다.

## 3. 사용법 (기능별)

- **대회 찾기**: `/race`에서 연·월·종목으로 필터링 → 러너블 직접 접수 대회는 `/competition`에서 "접수 중" 확인.
- **대회 상세/구매 단위**: 대회 카드 → `/comp`(대회 정보) → `/product`(종목·옵션·수량 선택, 접수/취소 기한 확인).
- **페이스 계산기**: 트레드밀 속도 입력 시 km당 페이스 자동 환산, 기록 변환기 제공.
- **커뮤니티**: 카테고리 선택 → 글 읽기 → 댓글·좋아요로 참여.
- **운세**: 본인인증 후 버튼 클릭 → AI 운세 → 커뮤니티 댓글로 "복채" 내는 재미 요소.

## 4. 유저 플로우

### (A) 가입/로그인
소셜 로그인 진입 → 마이페이지 접근 → 핵심 기능(기록·운세·대회접수) 이용 시 `/id-verification` 실명+휴대폰 본인인증(최초 1회).

### (B) 대회 접수
홈/대회일정/대회목록 → 대회 상세(`/comp`) → 상품·옵션 선택(`/product`) → (본인인증) → 결제 → 마이페이지 "대회 결제내역 / 참가정보 수정·취소"에서 관리.

### (C) 기록 관리
외부 기록조회(myresult) 또는 기록모아/레이스 로그 → 러너스 카드로 시각화.

### (D) 참여·리워드
커뮤니티 글·댓글 활동 → 바나나(포인트) 적립 → 경품 추첨 응모.

### (E) 콘텐츠/제휴 커머스
매거진·배너 → 제휴 상품(KB 마라톤 카드, 식품·기어 특가, 런트립)으로 연결.

## 5. 특징 및 관찰

**수익 모델 다각화**: 대회 접수 수수료, 스토어 커머스, 해외 런트립 여행상품, 금융 제휴(KB 마라톤 카드 모집 중개), 광고·리워드(바나나 경품)가 동시에 존재.

**제3자 대회 운영 리스크**: 커뮤니티에 'JTBC 서울마라톤' 주최 측 유동성 문제 관련 환불·정상개최 문의 글이 다수 게시됨. 플랫폼이 외부 대회 운영 리스크에 일부 노출됨(※ 사용자 게시글 기준, 사실 검증 별도 필요).

**본인인증 게이트**: 기록/운세/접수 등 핵심 기능 대부분이 휴대폰 본인인증을 전제로 함 → 실명 기반의 신뢰 가능한 대회 접수 플랫폼 지향.
